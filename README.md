# Squado

Sistema de gestão de equipes para líderes.

## Deploy

Push para `main` → deploy automático no Firebase Hosting via GitHub Actions.

## Arquivos

- `public/index.html` — Landing page (squado.com.br)
- `public/app.html` — Aplicação (squado.com.br/app)
- `firebase.json` — Configuração do Firebase Hosting
